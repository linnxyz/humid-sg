About HumidSG.io

HumidSG.io provides real-time relative humidity data from various weather stations across Singapore. This data is sourced from the Singapore Government Open Data Portal. For more details about the API and its usage, please refer to the official API documentation.

About This Project

HumidSG.io is a personal passion project by me, Linn. I created it to explore real-time data, specifically humidity levels, and to improve my web development, API, and UI design skills. This project is not affiliated with any government or organization. It’s just an experiment I built in my free time to learn how public APIs can be used in practical applications. Thanks for checking it out — I hope it helps you understand Singapore's humidity better!
